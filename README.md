# Exploring-Netflix-Content-Trends-Popular-Genres-and-Global-Distribution
Understand the distribution of movies and TV shows on Netflix.  Identify trends over years in Netflix content.  Analyze the popularity of genres and ratings.


## Project Overview
This project is an **interactive dashboard** for analyzing Netflix content data. Users can:
- Upload their own Netflix dataset
- Filter by content type, genre, year, or country
- Explore interactive charts
- Forecast future releases using ETS (Exponential Smoothing)
- Run SQL queries on the dataset

## Features
- Upload CSV or Excel data
- Interactive tables and charts with Plotly
- ETS forecasting for future Netflix releases
- Optional SQL query interface
- Filter by content type, genre, and year

## Technologies Used
- Python
- Streamlit
- Pandas
- Plotly
- statsmodels (ETS)
- SQLite (optional)
